一维数组的取值方式:
{$arr1[0]} -> {$arr1[1]} -> {$arr1[2]} -> {$arr1[3]}
{$arr1.0} => {$arr1.1} => {$arr1.2} => {$arr1.3}
二维数组的取值方式:
{$arr2[0][0]} -> {$arr2[0][1]}
{$arr2[1][0]} -> {$arr2[1][1]}
{$arr2.2.0} => {$arr2.2.1}
{$arr2.3.0} => {$arr2.3.1}

{$vol}
volist遍历二维数组:
{$vo} =
